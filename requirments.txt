--INSTALL THESE MODULES TO RUN THIS PROJECT--

    pip install discord.py
    pip install transformers
    pip install torch
    pip install sentencepiece

[What Each Module Does:]
discord.py → Allows interaction with the Discord API to send and receive messages.

transformers → Provides AI models like DistilGPT-2 for text generation.

torch → Required for running AI models efficiently.

sentencepiece → Helps in tokenization for better AI responses.
